# Instruções para IA / Codex

Sempre que alterar qualquer arquivo deste projeto Unreal Engine 5, registre a mudança no arquivo `AI_CHANGES.md`.

Cada registro deve conter:
- Data e hora
- Arquivos alterados
- O que foi feito
- Motivo da alteração
- Possíveis impactos ou cuidados
-OBS: nunca builde o projeto, eu que buildarei sempre

Formato:

## YYYY-MM-DD HH:mm
### Arquivos alterados
- Caminho/Do/Arquivo.ext

### Alterações
- Descrição objetiva do que foi modificado.

### Motivo
- Por que a alteração foi feita.

### Observações
- Riscos, próximos passos ou dependências.
